# comfyui_alikonfilms
A CLIP text encoder with 18 cinematic camera movement presets (dolly, pan, tilt, zoom, orbit and more). Subject-aware and free-camera modes, blend slider to mix your prompt with camera movements, and a live text output to preview the final prompt.

if Concatenate 0, Clip Text: enabled, Camera: disabled,
if Concatenate 1, Clip Text: disabled, Camera: enabled, 
if Concatenate 0.5, Clip Text: enabled, Camera: enabled,

Choose Camera Movement from either (0) or (1)
(0) subject aware will keep character in frame
(1) natural movement will move the camera as the name suggests regardless of the subject in frame
